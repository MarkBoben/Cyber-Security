Cybersecurity Toolkit Collection

A collection of practical cybersecurity tools developed for penetration testing, vulnerability analysis, cryptography, and system integrity monitoring.
This repository demonstrates the implementation of security concepts using Python.

The toolkit includes four major modules:

Penetration Testing Toolkit

Web Application Vulnerability Scanner

File Integrity Checker

Advanced Encryption Tool

These tools are designed for educational purposes, security research, and ethical penetration testing.

Project Overview
1. Penetration Testing Toolkit

A modular toolkit designed to assist in performing basic penetration testing tasks.

Features

Port Scanner

Brute Force Module

Network Enumeration

Service Detection

Technologies

Python

Socket Programming

Multithreading

Use Case

Used to identify open ports, running services, and weak authentication systems in a target environment.

2. Web Application Vulnerability Scanner

A security tool that scans web applications to identify common vulnerabilities.

Features

SQL Injection detection

Cross-Site Scripting (XSS) detection

URL parameter testing

Automated vulnerability reporting

Technologies

Python

Requests Library

BeautifulSoup

Use Case

Helps security professionals and developers identify vulnerabilities in web applications before attackers exploit them.

3. File Integrity Checker

A tool that monitors file changes by generating and verifying cryptographic hash values.

Features

SHA-256 hash generation

File change detection

Integrity verification

Monitoring of sensitive files

Technologies

Python

Hashlib Library

File System Monitoring

Use Case

Used to detect unauthorized modification of critical files, useful for system administrators and security analysts.

4. Advanced Encryption Tool

A cryptographic utility used to encrypt and decrypt files using strong encryption algorithms.

Features

AES-256 encryption

Secure file encryption

Decryption support

Key-based encryption

Technologies

Python

Cryptography Library

Use Case

Ensures confidentiality of sensitive data by protecting files with strong encryption.
